LinkedIn outreach follows a connection-first approach, not InMail-first. InMail is a one-shot Day 7 fallback only.

**Why:** 2026 benchmarks — blank connection requests hit ~45–68% accept, connection-with-note ~28–45%, InMails 10–25% reply (one-shot, no follow-up unless they reply). Notes trigger LinkedIn's sales-spam pattern-match on enterprise accounts. But founders engage more after an accept if there was a relevant note. And a typical B2B deal needs ~8 touchpoints — connection flow allows that, InMail doesn't.

**How to apply — prospect segmentation:**

**Enterprise C-suite → blank connection request (no note).**
- Fortune 500 / public-co execs, President of major division
- They get 50+ requests/day and flag noted ones as sales
- Blank looks organic — like you might already know each other
- After accept, DM with the real pitch

**Founders / startup CEOs → connection request with tight note (~200–280 chars).**
- Seed-to-late-stage founders
- Curiosity-driven, reward specificity
- Note earns both the accept AND higher post-accept reply rate

**Cadence (aligns with the 3-touch Day 0/3/7 rule in `outreach_cadence.md`):**
- **Day 0 (Touch 1):** Send connection (blank or noted per segment). If 1st-degree, send the DM directly.
- **Day 3 (Touch 2, after accept):** Post-accept DM with the actual pitch. Bump Attempt # in CRM.
- **Day 7 (Touch 3, final):** If still no reply, send the close-out DM — redirect ask or "circle back next quarter." If the connection never accepted, switch channel (InMail if Premium, or email fallback).

**InMail rules when used as fallback:**
- One shot, make it count
- Same humanization rules (sub-90 words, no em-dashes, sentence-case subject)
- Reference that you tried connecting first ("pinged a connect request last week, pivoting since we haven't crossed paths yet")

**Never do:** InMail as first touch for LinkedIn outreach. Burns the Premium allocation and reads transactional.

**Connection note format rules (for the noted ones):**
- Never start with the person's name — that reads as mail-merge. Start with the specific hook.
- Never end with "curious to connect" / "worth being connected" / "keen to connect" / "would value being connected." All AI tells.
- Drop the "I" at start of sentences when natural ("run partnerships at X" not "I run partnerships at X"). Humans truncate.
- End on the specific thing, not a generic connect ask. If there's a geographic or strategic overlap, name it. Otherwise stop.
- ≤200 chars ideal (LinkedIn max 300, but shorter reads more human).

**Post-accept DM style (confirmed working pattern):**
```
Hey {name}, I can see you're no longer at {old co}, but I think you'll be interested in what we're building, could I put 15 mins in your calendar?
```

Works even without the "no longer at" clause:
```
Hey {name}, I think you'll be interested in what we're building. Could I put 15 mins in your calendar?
```
