In all cold outreach, DMs, connection notes, emails, and partner-facing content, refer to the company as **{{YOUR_COMPANY}}**.

**Why:** Consistency across touchpoints. If some messages say "Acme" and others say "Acme Technologies Inc." or "Acme Labs" or an internal codename, prospects who check multiple touchpoints see incoherence or think they're different companies.

**How to apply:**
- All cold outreach (emails, LI InMails, LinkedIn DMs, connection notes, Twitter DMs) uses **{{YOUR_COMPANY}}**
- Email signatures: **`{{YOUR_NAME}} / {{YOUR_TITLE}}, {{YOUR_COMPANY}}`**
- Company description in prospect-facing copy: **"{{YOUR_COMPANY}} {{YOUR_COMPANY_ONE_LINER}}"**
- Internal notes / CRM descriptions can use shorthand — outbound uses the full consistent name
- If you have a codename used with close partners (common for stealth/early-stage cos), don't use it externally until you're ready

**Substitution guide:**
- `{{YOUR_COMPANY}}` → your public-facing company name
- `{{YOUR_COMPANY_ONE_LINER}}` → ~10-word description prospects will quickly understand
